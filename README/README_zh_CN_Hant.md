## 介紹

打造一組可以最少依賴、乾淨的开源框架，支援每個元件、檔案甚至程式碼段可以獨立使用，彼此沒有耦合。

## 開發語言

- Java JDK 1.8，若需使用更高版本環境，請移至 [Java JDK 17](https://gitee.com/gentkit/gentkit-for-java-jdk17)
